### **The Golden Rules of Particle Accounting**

Before we start, keep these conservation laws in your pocket. We check them for every single line:

1. **Charge ($Q$):** Must be equal on both sides.
    
2. **Baryon Number ($B$):** Protons and neutrons have $B=1$. Quarks have $B=1/3$. Mesons/Leptons have $B=0$.
    
3. **Lepton Number ($L$):** Specific to families (electron $L_e$, muon $L_\mu$). Particles are $+1$, antiparticles are $-1$.
    
4. **Strangeness ($S$):** Conserved in **Strong** and **EM** interactions, but can be violated in **Weak** interactions.

### **Detailed Solutions**

#### **(a)** $^{125}_{54}\text{Xe} + e^- \to ^{125}_{53}\text{I} + \nu_e$

- **Analysis:**
    
    - **Charge ($Z$):** Left is $54 + (-1) = 53$. Right is $53 + 0 = 53$. (Check $\checkmark$)
        
    - **Baryon Number ($A$):** Left is 125. Right is 125. (Check $\checkmark$)
        
    - **Lepton Number ($L_e$):** Left has an electron ($L_e = 1$). Right has an electron-neutrino ($L_e = 1$). (Check $\checkmark$)
        
- **Verdict:** **Allowed**
    
- **Interaction Type:** **Weak Interaction**.
    
    - _Professor's Note:_ This process is called **Electron Capture**. A proton inside the nucleus grabs one of its own orbiting electrons and turns into a neutron ($p + e^- \to n + \nu_e$). Since neutrinos are involved, it's a signature of the Weak force.
        

#### **(b)** $^{68}_{31}\text{Ga} \to ^{68}_{30}\text{Zn} + e^- + \bar{\nu}_e$

- **Analysis:**
    
    - **Charge ($Z$):** Left is $31$. Right is $30 (\text{Zn}) + (-1) (e^-) + 0 = 29$.
        
    - **The Problem:** $31 \neq 29$. Charge is missing!
        
- **Verdict:** **Not Allowed (Forbidden)**
    
- **Reason:** **Violation of Conservation of Charge.**
    
    - _Professor's Note:_ For this to work, the Gallium would need to emit a positron ($e^+$), not an electron ($e^-$). That is called $\beta^+$ decay.
        

#### **(c)** $^{235}_{92}\text{U} \to ^{141}_{56}\text{B} + ^{92}_{36}\text{Kr} + 3n$

- **Analysis:**
    
    - _Correction:_ The symbol "B" usually stands for Boron ($Z=5$), but the subscript says $56$, which is Barium ($\text{Ba}$). I will assume the paper has a typo and means Barium ($^{141}_{56}\text{Ba}$).
        
    - **Charge ($Z$):** Left is 92. Right is $56 + 36 + 0 = 92$. (Check $\checkmark$)
        
    - **Mass Number ($A$):** Left is 235. Right is $141 + 92 + 3(1) = 236$.
        
    - **The Problem:** $235 \neq 236$. We magically created a neutron out of nowhere!
        
- **Verdict:** **Not Allowed (Forbidden)**
    
- **Reason:** **Violation of Conservation of Baryon Number (or Mass Number).**
    
    - _Professor's Note:_ If there were only **2** neutrons ($2n$) instead of $3n$, this would be a valid nuclear fission reaction.
        

#### **(d)** $^{180}_{74}\text{W} \to ^{176}_{72}\text{Hf} + ^4_2\text{He}$

- **Analysis:**
    
    - **Charge ($Z$):** $74 \to 72 + 2 = 74$. (Check $\checkmark$)
        
    - **Mass ($A$):** $180 \to 176 + 4 = 180$. (Check $\checkmark$)
        
- **Verdict:** **Allowed**
    
- **Interaction Type:** **Strong Interaction (Nuclear)**.
    
    - _Professor's Note:_ This is standard **Alpha Decay**. The alpha particle ($^4\text{He}$) is held together by the strong nuclear force.
        

#### **(e)** $K^0 \to \pi^+ + \pi^-$

- **Analysis:**
    
    - **Particles:** Kaon ($d\bar{s}$) $\to$ Pion ($u\bar{d}$) + Pion ($d\bar{u}$).
        
    - **Charge:** $0 \to +1 - 1 = 0$. (Check $\checkmark$)
        
    - **Strangeness ($S$):**
        
        - $K^0$ contains an anti-strange quark $\bar{s}$. Strangeness $S = +1$.
            
        - Pions have no strange quarks. $S = 0$.
            
        - $\Delta S = 1$. Strangeness is **not** conserved.
            
- **Verdict:** **Allowed**
    
- **Interaction Type:** **Weak Interaction**.
    
    - _Professor's Note:_ Only the Weak interaction is allowed to change quark "flavors" (changing a strange quark into a down quark) and violate conservation of Strangeness.
        

#### **(f)** $n + n \to \Lambda^0 + K^0 + p + \pi^-$

- **Analysis:**
    
    - **Baryon Number:**
        
        - Left: $1 + 1 = 2$.
            
        - Right: $\Lambda^0(1) + K^0(0) + p(1) + \pi^-(0) = 2$. (Check $\checkmark$)
            
    - **Charge:**
        
        - Left: $0 + 0 = 0$.
            
        - Right: $0 + 0 + 1 - 1 = 0$. (Check $\checkmark$)
            
    - **Strangeness:**
        
        - Left: $0$.
            
        - Right: $\Lambda^0$ ($uds$, $S=-1$) + $K^0$ ($d\bar{s}$, $S=+1$). Total = $0$. (Check $\checkmark$)
            
- **Verdict:** **Allowed**
    
- **Interaction Type:** **Strong Interaction**.
    
    - _Professor's Note:_ Because Strangeness is conserved here (we created a strange particle and an anti-strange particle simultaneously, known as "associated production"), this proceeds via the Strong force, which dominates over the Weak force whenever it can.
        

#### **(g)** $\eta \to \gamma + \gamma$

- **Analysis:**
    
    - **Particles:** $\eta$ is a neutral meson. $\gamma$ is a photon.
        
    - **Charge:** $0 \to 0$. (Check $\checkmark$)
        
    - **Conservation:** All quantum numbers are conserved.
        
- **Verdict:** **Allowed**
    
- **Interaction Type:** **Electromagnetic Interaction**.
    
    - _Professor's Note:_ Whenever you see particles decaying purely into photons (gamma rays), it is almost always an Electromagnetic interaction.
        

#### **(h)** $\Lambda^0 \to p + \mu^- + \nu_\mu$

- **Analysis:**
    
    - **Charge:** $0 \to 1 + (-1) + 0 = 0$. (Check $\checkmark$)
        
    - **Baryon Number:** $\Lambda^0$ (Baryon) $\to p$ (Baryon). $1 \to 1$. (Check $\checkmark$)
        
    - **Lepton Number ($L_\mu$):**
        
        - Left: No leptons ($L_\mu = 0$).
            
        - Right: Muon ($\mu^-$ has $L_\mu = +1$) + Muon Neutrino ($\nu_\mu$ has $L_\mu = +1$).
            
        - Total Right $L_\mu = +2$.
            
    - **The Problem:** $0 \neq 2$. We created two leptons without creating any anti-leptons to cancel them out!
        
- **Verdict:** **Not Allowed (Forbidden)**
    
- **Reason:** **Violation of Conservation of Muon Lepton Number.**
    
    - _Professor's Note:_ If the reaction were $\Lambda^0 \to p + \mu^- + \bar{\nu}_\mu$ (with an **anti-neutrino**), it would work!
