$a \sim b$
It is the $\sim$ sign between two elements, showing that there is a relationship/equivalence there. A set of elements with this kind of relationship can be summed up together into a [[equivalence class]]. 
With the properties:
- **Reflexivity**$$a \sim a$$
- **Symmetry** $$a\sim b \Leftrightarrow b \sim a$$
- **Transitivity** $$a \sim b \land b \sim c \Rightarrow a \sim  c$$
