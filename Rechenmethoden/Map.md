A map is a way to connect elements from two sets, the **domain** $A$ and the **codomain $B$**. It is denoted by $$F: A \to B, \space a \mapsto F(a)$$
The map $F$ takes in the **argument** $a$ and gives the **value** or **image** $F(a)$.  The **image** is also the set of all possible "solutions" of the map $F(A) = \{ F(A) | a \in A \} \subseteq B$, which is part of the codomain, but doesn't have to be equal to that. If the image is the same as the codomain, then the map is **[[surjective]]**. Another case is if the map maps only one element from the codomain to an element from the domain (one to one mapping) and then it is **[[injective]]**. In the case that the map has both qualities, then it is **[[bijective]]**. 

# inverse map
If the map is bijective, it can be reversed and written as $$F^{-1} : B \to A$$
# composition (map)
This is the connection of two maps denoted by  $\circ$ $$G \circ F : A \to C, \space a \mapsto G(F(a))$$
As a special case there is the map of the [[Cartesian product]]  $$F: A \times B \to C, \space (a,b) \mapsto c = F(a,b)$$
