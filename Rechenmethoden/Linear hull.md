/Linear span
This is the set and at the same time a vector space with all possible linear combinations of elements of S, where S is: 
$$S \equiv \{\vec v_1, ..., \vec v_m\}; \space \vec v_i \in V$$

And the linear hull for that is: $$span(S) \equiv \{\vec v_1 a^1 + \vec v_2 a^2 + ... \vec v_m a^m| a^1, ..., a^m \in \mathbb{F} \}$$ where $\mathbb{F}$ is just any set of numbers. It can be the real numbers, complex or any of similar kind.
