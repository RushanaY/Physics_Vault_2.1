If you can't represent a vector using a linear combinations of other vectors, then those are linearly independent. Therefore we can represent a vector with a linear combination of other vectors only if those are linearly dependant to each other. 

To test linear dependency there are multiple ways of doing that: 
1. If vectors are **linearly dependant**, then there exist non vanishing coefficients
	Multiplying every vector with a coefficient $a_i$ and adding them up to sum up to 0, we create a [[System of linear equations]] and solve it. If in the end there is an equation can be solved only by making all coefficients equal to 0. $$\vec v_1 a^1 + \vec v_2 a^2 = 0$$$$a^1 = a^2 = 0$$ This means that there are infinitely many possible combinations of coefficients.
