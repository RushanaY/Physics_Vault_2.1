This is a "container" that can hold objects = [[Element]]s. 
A set $A$ with elements $a,b,c,...$ is denoted by $$A = \{a,b,c...\}$$
In order to indicate the relationship between set and element, we denote "a is element of set A" by writing $$a \in A$$
The number of elements in the set is called a [[cardinality]] 
A set can have any amount of elements, which don't have to be listed. Thereby there are the **countable** and **uncountable** sets. We can show that a set is countable, if every element in this set can be paired with a natural number in a bijective relationship. Where as if this is not possible, it is uncountable with the example being the [[real numbers]] 
# Rules 
There are different kinds of sets and kinds to order them. The **empty set** has no elements and is denoted by $A = \{\}$ or $A = \emptyset$. A **subset** is part of a bigger set. To show that $B$ is a subset of $A$ we write $$B \subset A$$ or more explicitly $$\{a,b\} \subset \{a,b,c\}$$In the rare case that the subset is equal to the set. this can be more precisely written as $$B \subseteq A$$On the contrary, if this should not be the case $$B \subsetneq A$$
Sets can be added together into a **union** $$A \cup B$$or "subtracted" in an **intersection** $$A \cap B$$
If we don't want a specific element to be in the set it is shown in the **difference** $$A / \{c\}$$
Set can be organized by using [[equivalence class]]es by writing $$a \sim b$$This means that two sets have a relationship that is characterized by a criterium. For example Vanessa and Kate are equivalent under the criterium of being both female.  

# Notation
Just looking at the notation of the set, one can determine what kind of elements will be there and under what condition. General notation from [[Cantor's set theory]] is:
$$set \space name = \{elements| rule \space or \space condition\}$$
