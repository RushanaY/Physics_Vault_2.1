==this is a way to create a sigma algebra== 
1. make a premeasur on a ring or and algebra -> gives the "correct size"
2. define outer measure on all subsets, that would cover the chosen ring/algebra ->gives the size of all sets 
3. declare a set $E \subseteq X$ that is **caratheodory measurable** if the following holds $\mu^A( = \mu^* *A \cap E) + \mu^* (A \cap E^C)$ -> picks out all the sets that behave nicely and are measurable 
4. out all the nice sets in our finished measure
