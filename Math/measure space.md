It is a space equipped with a [[measure]] $(X,A, \mu)$ 

we call it [[complete]], when every [[null set]]is inside the algbera $\mathcal{A}$ 


# Application 
If we have a measure space and $\mathcal{N} : = \{ Y \subseteq X :\mu^* (Y) =0 \}$, then there is a [[sigma algebra]]
$$\overset{\sim}{\mathcal{A}} : = \{ A \cup N: A \in \mathcal{A}, N \in \mathcal{N}\}$$
and we define a [[complete]] [[measure]] with the helo of $\overset{\sim}{\mu} (A \cup N): = \mu(A)$  

We can complete the measure to be $(X, \overset{\sim}{\mathcal{A}}, \overset{\sim}{\mu})$
