**Induction**
1. Inductionsanfang : show $P(1)$
	- show the expression for the first element of the used set 
2. Induktionsschritt: show $\forall n \in \mathbb{N} :P_n \Rightarrow P(n+1)$ 
	- assume that $P(n)$ holds and it is yet to be proven for the next element
	- the assumption is that the expression holds for n and for any next one
