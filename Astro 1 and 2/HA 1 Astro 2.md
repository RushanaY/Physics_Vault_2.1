# a modell of the universe expansion
using $F = - \frac{GMm}{R^2}$ and $M = \frac{4 \pi}{3} R^3 \rho (t)$ 
=> $R''(t) = - \frac{4 \pi G}{3} \rho(t) R(t)$ 

using the ansatz $R(t) = R_0 (\frac{t}{t_0})^{2/3}$ 
taking the double derivative gives: $R''(t) = - \frac{GM}{R_0^2} (\frac{t_0}{t})^{4/3}$ 
=> $\frac{2R_0}{9t_0^2} = \frac{GM}{R_0^2} \Rightarrow t = \sqrt{\frac{2R_0^3}{9GM}}$
